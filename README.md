# TapPay — Legal & Policy

> Public policy documents for the TapPay application, including the Privacy Policy and related terms.

## Overview

This repository hosts the legal and policy pages for **TapPay**. The documents are published as static HTML so they can be linked directly from the app, app stores, and support pages.

## Contents

- [`docs/index.html`](docs/index.html) — landing / policy index
- [`docs/privacy.html`](docs/privacy.html) — Privacy Policy

## Hosting

The `docs/` folder is structured for static hosting (e.g. GitHub Pages). Once Pages is enabled for this repository, the policy will be available at your Pages URL.

## Viewing Locally

```bash
git clone https://github.com/Ferousco-dev/TapPay-policy.git
cd TapPay-policy
python3 -m http.server 8000
# open http://localhost:8000/docs/
```
